# Настройка фонового изображения

## Как добавить фоновое изображение

1. **Поместите ваше изображение океана** в папку `assets/images/` (создайте папку если её нет):
   ```
   assets/
   └── images/
       └── ocean_background.jpg  # или .png
   ```

2. **Обновите путь к изображению** в файле `src/components/OceanBackground.tsx`:
   ```typescript
   // Замените эту строку:
   source={{ uri: 'ocean_background' }}
   
   // На одну из этих опций:
   
   // Опция 1: Локальное изображение
   source={require('../../assets/images/ocean_background.jpg')}
   
   // Опция 2: Изображение из интернета
   source={{ uri: 'https://your-domain.com/ocean_background.jpg' }}
   ```

3. **Для iOS** - добавьте изображение в Xcode проект:
   - Откройте `ios/CallOcean.xcworkspace` в Xcode
   - Перетащите изображение в папку `Images.xcassets`
   - Убедитесь, что изображение добавлено в `AppIcon.appiconset`

4. **Для Android** - поместите изображение в папку:
   ```
   android/app/src/main/res/drawable/ocean_background.jpg
   ```

## Рекомендации по изображению

- **Размер**: 1080x1920 пикселей (Full HD)
- **Формат**: JPG или PNG
- **Содержание**: Подводная сцена с синими тонами, лучами света, кораллами
- **Оптимизация**: Сожмите изображение для быстрой загрузки

## Альтернативные варианты

Если у вас нет подходящего изображения, можете использовать:

1. **Градиентный фон** (уже настроен как fallback)
2. **Анимированный фон** с CSS анимациями
3. **Видео фон** для более динамичного эффекта

## Тестирование

После добавления изображения:
1. Запустите `npm run ios` или `npm run android`
2. Проверьте все экраны приложения
3. Убедитесь, что изображение отображается корректно
4. Проверьте производительность на разных устройствах

## Отладка

Если изображение не отображается:
1. Проверьте путь к файлу
2. Убедитесь, что файл существует
3. Проверьте права доступа к файлу
4. Посмотрите в консоли на ошибки загрузки изображений
